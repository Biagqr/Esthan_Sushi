# Notas originais

Estes arquivos são as anotações e trechos de código originais do projeto,
mantidos aqui apenas para referência histórica. A aplicação real vive na raiz
do repositório (Vite + React + Tailwind).

- `Cardapio digital` — rascunho da estrutura de pastas e dados do menu.
- `codigo da pagina` — primeira versão da página inicial.
- `estrutura profissional` — planejamento da arquitetura completa.
- `rota` — rascunho das rotas.
- `npm create` / `code` — comandos usados na criação do projeto.
